Intel Support Mobile App

The Intel® Support App helps its users browse, search and compare product specifications with easy access and offline functionality. Users can create and track support tickets, and communicate directly with Intel using the support channels.

Project Timeline

10 months

Role

UX/UI Designer

Tools Used

Adobe XD, Invision, Adobe Illustrator, Adobe Photoshop

Project Type

Tech Support, UX Research, UI Design, UX Design

Objectives

  • Grow user retention Increase user satisfaction
  • Update app to meet Intel standards
  • Generate stronger
  • Customer reliance
  • Decrease errors and user dropoff

Final Designs

The benefit of a product that is already out is that there are many ways to get user feedback and try to understand the users of the product. Where they find themselves most frustrated with the product, what are the most common issues and what is enjoyed or used frequently. After running through the feedback gathered and passing through many competitive analyses, I was able to start brainstorming and sketching out ideas that would later be reviewed with my project manager and stakeholders to see if they were inline with the user and business goals.

  • Downtime Messages
  • 2D Matrix Barcode Scanner
  • Updates Indicator
  • Pull To Refresh
